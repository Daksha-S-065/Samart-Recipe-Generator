# Smart Recipe Generator

->Description
Smart Recipe Generator is a web application that suggests recipes based on ingredients selected by the user.

It also shows related YouTube cooking videos to help users follow the recipe.


->Features

- Ingredient-based recipe search  
- Recipe suggestions using database  
- YouTube video integration  
- Simple and user-friendly interface  


->Technologies Used

Frontend:
- HTML
- CSS
- JavaScript

Backend:
- JavaScript

Database:
- MongoDB

API:
- YouTube Data API

Tools:
- VS Code


->How It Works

1. User selects ingredients  
2. System searches recipes in database  
3. Matching recipes are shown  
4. Related YouTube videos are displayed
